NDVI Calculation & Standards

Normalized Difference Vegetation Index (NDVI) is calculated using satellite imagery:

NDVI = (NIR - RED) / (NIR + RED)

Data Sources: Sentinel-2 (10-20m resolution), Landsat 8/9 (30m resolution), MODIS (250-500m resolution)

Verification Criteria

When reviewing plantations, ensure:

  • NDVI Range: 0.3-0.8 for healthy vegetation (avoid values >0.9 which may indicate stress)
  • Consistency: Check for spatial uniformity across plantation area
  • Seasonal Variation: Account for expected NDVI changes by season and crop type
  • Image Quality: Verify cloud-free, recent imagery (<30 days old)
