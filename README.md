
# 🌱 Carbon Credit Marketplace
### A Next.js Carbon Credit Marketplace for Farmers, Businesses, and Admins

This repository contains a modern full-stack carbon credit marketplace implemented in `nextjs-app/`.
The app connects farmers, businesses, and administrators through role-based dashboards, NDVI-based plantation verification, and an interactive Leaflet map.

---

## 🚀 Quick Start

```bash
cd nextjs-app
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## ✨ What’s Included

- **Next.js 14** with App Router
- **Tailwind CSS** styling
- **Leaflet + React Leaflet** map visualization
- **Role-based auth** for farmer, business, and admin flows
- **API routes** for plantations, purchases, verification, and stats
- **NDVI education** and verification guide
